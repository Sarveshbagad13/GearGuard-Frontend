# GearGuard Workspace

This workspace now has two app folders:

- `GearGuard-frontend/` for the React + Vite frontend
- `GearGuard-backend/` for the Node.js + Express backend

## Run

Frontend:

```bash
cd GearGuard-frontend
npm install
npm run dev
```

Backend:

```bash
cd GearGuard-backend
npm install
npm run dev
```
